# dtpm-util

Funcionalidades para cálculos y procesamientos en el DTPM

## Instalación

Desde la linea de comando 

```
pip install git+ssh://git@github.com/jomavera/dtpm_util.git
```

Luego se puede importar el módulo `dtpm_util` en Python.

## Requisitos

- numpy
- pandas = 2.0.2
- openpyxl
- xlrd
